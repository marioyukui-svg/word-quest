# 🎮 Word Quest · 单词大冒险

> 玩着玩着就记住了——让背单词变成一场冒险！

一款专为孩子设计的英语单词游戏，基于 **英语高频5000词** 制作。打开浏览器就能玩，无需安装、联网。

---

## 🎯 游戏模式

| 模式 | 玩法 | 适合场景 |
|------|------|----------|
| 🃏 **记忆配对** | 翻牌配对——把英文单词和它的中文释义配成一对 | 新词初学、视觉记忆型孩子 |
| ⚡ **极速抢答** | 看到英文单词，从4个选项里选出正确中文释义，限时10秒 | 复习巩固、反应力训练 |
| ✍️ **拼写达人** | 看中文释义和例句，拼写出正确的英文单词 | 拼写训练、书写巩固 |

## 🌟 特色

- **898 个高频单词**，分8个阶段逐步推进
- **计分+连击系统**，激励孩子持续挑战
- **音效反馈**（轻快提示音），每个操作都有回应
- **🎊 彩花特效**，高分通关有庆祝动画
- **移动端适配**，手机/平板也能玩
- **进度保存**，下次打开接着玩
- **完全离线可用**，部署后无需网络

## 🚀 快速使用

### 方式一：直接打开（推荐）

访问 GitHub Pages 链接：
```
https://你的用户名.github.io/word-quest/
```

### 方式二：本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/word-quest.git

# 用浏览器打开 index.html 即可
open word-quest/index.html
```

## 📂 项目结构

```
word-quest/
├── index.html      # 游戏主页面（含全部样式和逻辑）
├── words.js        # 单词数据（898个高频词）
├── README.md       # 本文件
└── parse_words.py  # 数据解析脚本（仅首次生成用）
```

## 🛠️ 技术栈

纯前端，零依赖：
- 原生 HTML + CSS + JavaScript
- Web Audio API（音效）
- localStorage（进度存储）
- 响应式设计（Flex/Grid）

## 📝 自定义

想用自己的单词表？修改 `words.js` 中的 `WORD_DATA` 数组即可，格式：

```javascript
{
  word: "example",        // 英文单词
  meaning: "例子",        // 中文释义
  sentence: "This is an example.",  // 例句
  translation: "这是一个例子。"      // 例句翻译
}
```
